A web-based demo for real-world type system

For a easier understanding of how to operate our tool, a quite web-based live demo is created here: webRWT.

The webRWT is a simplified version of our tool, it does the following:
(1) allows users to select a programming language;
(2) allows users to modify the programs;
(3) Shows examples of real-world types;
(4) allows users to create real-world type rules;
(5) allows users to create real-world type bindings;

Analysis techniques provided by webRWT

The webRWT demonstrates some of the analysis techniques provided by real-world type system: (1) units checking of the program ;
(2) real-world constraint checking;
(3) Reasonableness and range analysis;
The principles and implementation details about the analysis techniques can be found in our documentation.
